# Bevzine Full Stack

Premium beverage guide + price finder + community platform.

## Tech Stack
- Next.js 14 (App Router)
- Prisma ORM
- PostgreSQL
- NextAuth
- TailwindCSS

## Features
✅ Explore Drinks  
✅ Brands & Variants  
✅ State-wise Price Finder  
✅ Guides  
✅ Pairings  
✅ BevMedia Community  
✅ BevMed Health Awareness  
✅ Membership Page  
✅ Shop/Deals Affiliate Page  
✅ Admin Dashboard Starter

```bash
npm install
